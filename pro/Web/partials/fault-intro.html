Peach monitors your target while fuzzing to detect faults (checking for irregular behavior such as a program crashing or remote device availability).

This portion of the wizard will set up the Pit to perform fault detection of the target you are fuzzing.

You'll need the following information to complete the Pit Fault Detection section:

  • Whether you want to perform fault detection on the local machine or a remote machine.
  • If the detection is done on a remote machine, the remote machine's hostname/IP name and OS
  • If performing fault detection on a process, information will be asked about starting the process or service
