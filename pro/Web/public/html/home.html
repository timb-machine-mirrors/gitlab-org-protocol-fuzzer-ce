Welcome to the Peach Fuzzer ® home page. This page appears when you start Peach from the command prompt.

A list of the 10 most recent fuzzing sessions (jobs) makes up the main content on this page.

For any entry, you can perform the following actions:

  • Click the icon to view the report generated for the fuzzing session.
  • Click the icon to delete the job entry, thereby removing the corresponding report files from disk.

What do the menu entries on the left side of the page do?

Home
Navigate to this page.
Library
View and select a fuzzing definition to configure for running as a job.
Jobs
View a list of all fuzzing sessions that have run on this computer.
Help
Engage online help for Peach.
Forums
Get support from the online Peach Forums.
