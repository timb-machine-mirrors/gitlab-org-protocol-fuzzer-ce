== SHA256Fixup

Performs a SHA-256 hash of _ref_ element's value and places it in the parent element.
The SHA256 algorithm is defined in US Federal Information Processing Standard (FIPS) PUB 180-4 and is implemented in .Net Framework Implementation - System.Security.Cryptography.
This is used when fuzzing protocols that hash data with SHA-256.

=== Parent Elements

 * xref:String[String]
 * xref:Blob[Blob]
 
=== Parameters

 * xref:ref[ref] -- Reference to element whose value will be hashed
 
 
=== Examples

[source,xml]
----
<Peach xmlns="http://peachfuzzer.com/2012/Peach" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="http://peachfuzzer.com/2012/Peach ../peach.xsd">

	<DataModel name="Packet">
		<Block name="Header">
			<Number name="SrcPort"  size="16" endian="big" value="1234"/>
			<Number name="DestPort" size="16" endian="big" value="1235"/>
			<Number name="Length"   size="16" endian="big">
				<Relation type="size" of="Packet"/>
			</Number>
		</Block>
		<Blob name="Checksum">
			<Fixup class="SHA256Fixup">
				<Param name="ref" value="Header" />
			</Fixup>
		</Blob>
	</DataModel>

	<StateModel name="TheState" initialState="initial">
		<State name="initial">
			<Action type="output">
				<DataModel ref="Packet" />
			</Action>
		</State>
	</StateModel>

	<Test name="Default">
		<StateModel ref="TheState"/>
		<Publisher class="ConsoleHex" />
	</Test>
</Peach>
----
