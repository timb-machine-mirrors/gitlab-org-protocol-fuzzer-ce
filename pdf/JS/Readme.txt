=================

Copyright (c) Adobe Systems, Inc.
Michael Eddington

To start this fuzzer, follow these steps:

 1. Verify the folder "logs" does not exist
 2. Edit "template.xml" and verify Acrobat path
 3. Verify c:\peach exists with valid Peach runtime
 4. From a command line run "python runfuzzer.py"
