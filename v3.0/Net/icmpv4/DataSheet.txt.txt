Protocol
--------

 * Test Suite: ICMPv4
 * Direction: Client, Server(TODO)
 
ICMPv4 protocol is message relay protocol on top of IPv4.
It provides error responses for the IPv4 protocol as well as simple
data transmision. It is commonly used in the ping application for 
testing network response and connectivity.
