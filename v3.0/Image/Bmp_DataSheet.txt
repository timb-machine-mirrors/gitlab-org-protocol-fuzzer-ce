:Doctitle: BMP Peach Pit Data Sheet
:Description: BMP Image Format

 * Peach Pit: BMP
 * Supported Platorms: Windows, Linux, OS X

Bitmap is a Microsoft defined file format to contain raster graphic images.
The file contains fixed sized headers and a variable length pixel array.
The BMP header varies by a variety versions and each version successivly appends fields to its previous version.
The file format is closely tied to the DIB internal data structure in the Windows API.
