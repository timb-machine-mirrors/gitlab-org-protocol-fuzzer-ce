Pit Configurator

This wizard will let you configure your Peach Professional pit files. Pit files contain all the information Peach needs to fuzz your files.

After you complete each part of this wizard, the pit you create will be able to:
  • Set variables
  • Detect Faults
  • Tell Peach where to collect the generated output data
    • Specify how you want Peach to fuzz your files.
  • Test the pit with the configuration
  • Save the work for later loading
  • Do it all in a visual manner as opposed to editing Pit files
NOTE: avoid using Peach specific tech language such as "Monitors" and "Agents"
