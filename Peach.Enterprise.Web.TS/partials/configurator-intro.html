Pit Configurator

Purpose of the Configurator is for configuring pit files included with Peach Professional to:

  • Select Monitors (Fault Detection, Data Collection, and Automation) to apply to the pit.
  • Test the pit with the configuration
  • Save the work for later loading
  • Do it all in a visual manner as opposed to editing Pit files
NOTE: avoid using Peach specific tech language such as "Monitors" and "Agents"
