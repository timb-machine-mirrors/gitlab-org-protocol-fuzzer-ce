Pit Configurator

This wizard will let you configure a Peach Pit (fuzzing definition) from the Peach Pit Library. Pit files contain all the information needed to perform fuzzing of a target. After you complete each part of this wizard, the pit will be fully configured and ready to run.

This wizard will guide you through:

  • Configuring the pit variables
  • Configuring fault detection
  • Configuring data collection
  • Configuring target automation
  • Testing the pit to verify the configuration is correct
  • Save the work for later loading
