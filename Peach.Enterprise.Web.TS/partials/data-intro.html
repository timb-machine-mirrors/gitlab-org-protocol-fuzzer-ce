Data Collection

Data Collection section of the Pit Configurator is a series of questions and answers that helps the user choose the appropriate Peach Monitors for collecting resulting data from Peach.
The user may go through this process multiple times to select multiple Data Collecting monitors.
At the end of the Q-A the total list of Monitors chosen by the Pit Configurator will be displayed.
The user will then have the option to:

  • Delete any of the resulting Monitors
  • Begin the Q-A process again to choose more Monitors
  • Move on to Automation
