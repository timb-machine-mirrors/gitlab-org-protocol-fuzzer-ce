Peach monitors your files while they are fuzzed (checking for irregular behavior such as a program crashing or remote device availability) to detect faults and save relevant information in the logging files.
This portion of the wizard will set up the Pit Fault Detection section. You'll only have to do this once for each Pit you create.
You'll need the following information to completer the Pit Fault Detection section:

  • Whether you want to detect a fault on a local or remote machine
  • If the detection is done on a remote machine, the remote machine's hostname/IP name and OS
  • System and file information about your fault detection machine
  • What you want Peach to do when it detect target fault conditions
