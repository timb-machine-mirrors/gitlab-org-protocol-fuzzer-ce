Set Variables - Introduction

The Pit Set Variables section allows you to change the variables in the Pit Configuration file.
The variables you will need to set depend on the pit you are configuring. Because the variable list is pit installation dependent, you will need to set different variables for different pit installs.
There are some common pit variables that you will need to set no matter which pit you are configuring:

  • The path to the pit library where Peach will log any faults
  • Mutation strategy for Peach to use while fuzzing. Peach has a default strategy but if you wish, you can choose from a s predefined list of available strategy options.

Depending on the the pit to be configured, you may be setting variables with the following information:
  • IP address
  • MAC address
  • Target process name
  • Additional information about Target
